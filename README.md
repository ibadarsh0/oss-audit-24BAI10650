## 📜 Script Descriptions

### script1_system_identity.sh — System Identity Report
Displays system information such as Linux distribution, kernel version, user details, uptime, and license information.

### script2_package_inspector.sh — FOSS Package Inspector
Checks whether a given package is installed, shows its details, and prints a short description using a case statement.

### script3_disk_auditor.sh — Disk and Permission Auditor
Loops through important directories and reports their permissions, ownership, and disk usage.

### script4_log_analyzer.sh — Log File Analyzer
Reads a log file, counts occurrences of a keyword, and displays the last matching lines.

### script5_manifesto_generator.sh — Open Source Manifesto Generator
Takes user input and generates a personalized open-source philosophy statement saved to a file.
